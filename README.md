A workflow for preparing and training from an audio + transcript dataset for speaker classification

---

## Download Canned CDP Data
```bash
pip install speakerbox[data]
python scripts/download_cdp_dataset.py seattle
```

## Installation
`pip install git+https://github.com/CouncilDataProject/speakerbox.git`
